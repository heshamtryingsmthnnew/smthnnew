So this is a tool to like help engineering students with coursework and other stuff, its basically a workflow tool, the goal is to get you the answer in the cleanest most efficient way possible.
Im currently doing thisw sindependently so expect things to move slowly as i make changes and adjust things
The main defining feature of this tool is the verification logic:
Unlike regular Ai tools that have a chance to make mistakes, this tool cross checks answers with a independent algebra engine to make sure there are no mistakes.

Tech stack (roughly):
- Frontend: React / Next.js  
- Backend: Node.js / Express  

This project is far from complete and is still in a very barebones state, but i hope in the near future as i work on it and improve itll be something worth making public.
